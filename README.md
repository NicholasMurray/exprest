# Exprest

An example of multiple domains served by express using an api hosted on one of those domains

## Run the express servers

`node expressapp.js`

## Test URLS

### The api served from port 3000

`http://localhost:3000/api`

### Blue server URL

`http://localhost:3000`

### Green server URL

`http://localhost:3001`

### Red server URL

`http://localhost:3002`
